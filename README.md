# nocodeofconduct.dev

Landing page for `https://nocodeofconduct.dev`, built with Bun, Astro, and Primer Brand UI.

## Development

Install dependencies:

```bash
bun install
```

Start the Astro dev server:

```bash
bun run dev
```

Run the test suite:

```bash
bun run test
```

Build the static site:

```bash
bun run build
```

The build verifies the generated `dist/` output afterward using Bun-native file I/O and globbing.

Preview the production build locally:

```bash
bun run preview
```
